# Maly User Manual

## Welcome to Maly

Maly is a social networking platform that connects people through local events, meaningful connections, and AI-powered discovery. This guide will help you navigate all the features of the app.

---

## Getting Started

### Creating Your Account

1. **Open the App**: Navigate to the Maly website or open the mobile app
2. **Tap "Get Started"**: Begin the signup process
3. **Enter Your Details**:
   - Choose a unique username
   - Enter your email address
   - Create a secure password
   - Add your full name
4. **Upload Profile Photos**: Add up to 5 photos to your profile
5. **Set Your Location**: Select your current city
6. **Choose Your Vibe**: Select moods that describe you (e.g., Adventurous, Creative, Social)
7. **Set Your Intention**: What are you looking for? (Dating, Social, Networking, Friends)
8. **Complete Setup**: You're ready to explore!

### Logging In

1. Go to the login page
2. Enter your username and password
3. Tap "Login"
4. You'll be taken to the Discover page

## Main Features

### Discover Events

The **Discover** page is your gateway to local events.

**Finding Events:**
- Browse events in your city
- Use filters to narrow results:
  - **Category**: Music, Art, Food, Tech, Social, Workshop
  - **Date**: Today, This Weekend, This Week, This Month
  - **Price**: Free, Paid
- Tap any event card to see full details

**Event Details Page:**
- View event description and photos
- See the event schedule/itinerary
- Check the dress code (if any)
- View the venue on a map
- See who's attending

### Attending Events

**Free Events:**
1. Open the event
2. Tap "Interested" or "Attending"
3. You're added to the guest list

**Paid Events:**
1. Open the event
2. Tap "Buy Ticket"
3. Select ticket quantity
4. Complete payment via Stripe
5. Receive your ticket with QR code

**RSVP Events:**
1. Open the event
2. Tap "Request to Attend"
3. Add an optional message to the host
4. Wait for the host to approve your request
5. Once approved, you'll be added to the event group chat

### Creating Events

**Prerequisites:**
- For paid events, you must set up Stripe Connect to receive payments

**Creating an Event:**
1. Tap the "+" button in the navigation
2. Choose your event type:
   - **Free**: No ticket required
   - **Paid**: Set a ticket price
   - **RSVP**: Require approval to attend
3. Fill in event details:
   - Title and description
   - Date and time
   - Location and address
   - Category
   - Capacity (optional)
   - Dress code (optional)
4. Add event itinerary/schedule
5. Upload cover image
6. Tap "Create Event"

**Managing Your Events:**
- View attendees and interested users
- Approve or decline RSVP requests
- Edit event details
- Delete your event

### Connect with People

The **Connect** page helps you discover and connect with other users.

**Browsing Users:**
- View profiles of people in your city
- Filter by:
  - **Gender**: All, Men, Women
  - **Intention**: Dating, Social, Networking, Friends
  - **Vibe**: Various mood filters

**Viewing Profiles:**
- Tap a user card to see their full profile

- See their current mood/vibe

**Connecting:**
- Tap the contact/message icon to start a conversation

### Messaging

The **Inbox** page contains all your conversations.

**Types of Conversations:**
- **Direct Messages**: One-on-one chats with other users
- **Event Group Chats**: Automatically created when you're approved for RSVP events

**Sending Messages:**
1. Open a conversation
2. Type your message
3. Tap send
4. Messages are delivered in real-time

**Starting a New Conversation:**
1. Go to a user's profile
2. Tap the message icon
3. Start chatting

The **Concierge** (chat bot) helps you discover events using natural language.

**How to Use:**
1. Tap the Concierge icon in navigation
2. Ask questions like:
   - "What events are happening this weekend?"
   - "Show me music events in Miami"
   - "Any free art exhibitions today?"
   - "What's going on tonight?"

**The AI will:**
- Understand your request
- Search available events
- Recommend matching events
- Answer follow-up questions

## For Event Hosts

### Setting Up Payments

To receive payments for paid events:

1. Go to Settings > Payment Settings
2. Tap "Get Paid for Events"
3. You'll be redirected to Stripe
4. Complete Stripe onboarding:
   - Enter personal information
   - Add bank account for deposits
   - Verify identity
5. Return to Maly
6. Your account is now ready to accept payments

**Platform Fee:**
- Maly charges 3% on each ticket sale
- Stripe charges additional processing fees (~2.9% + $0.30)
- You receive the remainder directly to your bank

### Managing RSVPs

For RSVP-required events:

1. Go to your event
2. Tap "Manage Applications"
3. View pending requests
4. For each request:
   - Read their message
   - View their profile
   - Tap "Approve" or "Decline"
5. Approved users are automatically added to the event group chat

### Event Group Chat

When you approve attendees:
- A group chat is automatically created
- All approved attendees can see and participate
- Use it to share updates, logistics, and connect attendees
- You (the host) are automatically included

## Troubleshooting

### Can't Log In

- Check your username and password
- Try resetting your password
- Clear browser cache and try again
- Contact support if issues persist

### Payment Failed

- Verify your card details are correct
- Ensure sufficient funds
- Try a different payment method
- Check with your bank for any blocks

### Not Receiving Notifications

1. Check notification settings in the app
2. Verify device notification permissions
3. Ensure you've subscribed to push notifications
4. Check your internet connection

### Event Not Appearing

- Confirm the event is in your selected city
- Check date filters
- Refresh the page
- The event may have been canceled or deleted

### Profile Photos Not Uploading

- Check file size (max 5MB per image)
- Use supported formats (JPG, PNG, GIF, WebP)
- Check internet connection
- Try a different image
